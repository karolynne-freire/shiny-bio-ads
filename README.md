
# Portfólio Acadêmico e Profissional | Karolynne

Este repositório contém o código-fonte do meu portfólio pessoal, desenvolvido como parte do desafio prático da **Digital Innovation One (DIO)**. O projeto tem como objetivo centralizar minhas principais atividades acadêmicas, projetos de software e minha atuação na pesquisa científica.

## 🚀 Destaques e Atuação

  * **Formação:** Graduanda em Análise e Desenvolvimento de Sistemas (ADS) pelo **IFPE - Campus Jaboatão dos Guararapes**.
  * **Pesquisa:** Bolsista de Iniciação Científica (**PIBIC**), atuando no projeto **Sunflower** (Pesquisa voltada ao apoio emocional).
  * **Desenvolvimento:** Autora do projeto **Minerva Lib**, focado em automação de processos de gestão bibliotecária.

## 💻 Tecnologias Utilizadas

Para esta versão, optei pelo uso de tecnologias *vanilla* para demonstrar domínio nos fundamentos de Front-end:

  * **HTML5:** Estruturação semântica de dados.
  * **CSS3:** Layout responsivo, Flexbox e estilização avançada.

## 📈 Evolução do Projeto (Roadmap)

Este projeto está em sua fase inicial (V1). O cronograma de atualizações futuras inclui:

  - [ ] Refatoração e migração para o framework **Next.js**.
  - [ ] Implementação de componentização com **React**.
  - [ ] Integração com a **API do GitHub** para exibição dinâmica de repositórios.
  - [ ] Otimização de performance e acessibilidade (WCAG).
